# Roadmap

## Milestone 0: Go scaffold
- Go module, CLI entrypoint, config loader, session detection stub

## Milestone 1: Session detection
- Gateway connectivity probe with clear status reporting
- Safe idle recheck loop

## Milestone 2: Python integration
- Bridge to Python for market data ingestion and strategy/model execution

## Milestone 3: Strategy runtime
- Strategy registry wiring, lifecycle management, extensible plugins

## Milestone 4: Observability and resilience
- Structured logging, metrics hooks, retry policies

## Milestone 5: Packaging and release
- Build artifacts and release automation
